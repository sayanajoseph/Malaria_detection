# Chronic Kidney Disease Detection using Machine learning
## Overview
This project focuses on the detection of Chronic Kidney Disease (CKD) using machine learning techniques. The model analyzes medical parameters to classify whether a patient is affected by CKD, helping in early diagnosis and decision support.
## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Dataset
Medical dataset used for Chronic Kidney Disease detection (public dataset).

## ML Approach
- Supervised Learning
- Classification Model

## Results
The model predicts the presence of Chronic Kidney Disease with reasonable accuracy.
